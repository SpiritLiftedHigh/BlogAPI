# BlogAPI
个人博客的接口，可以直接下载进行练习


<h2>所有的路由接口</h2>

>/bloglist   // 跳到文章列表页
>
>/doLogin  // 验证登录信息
>
>/doRegister  // 验证注册信息
>
>/submitBlog   // 提交博客信息
>
>/blog/:id  // 获得某条博客信息
>
>/updateLike  // 更新博客的喜欢数
>
>/submitReview  // 提交博客的评论
>
>/submitReply  // 提交用户之间的回复
>
>/messlist   // 获得所有的消息列表
>
>/usercenter   // 获得个人的详细信息
>
>/myblog/:id  // 获得某位用户的所有博客
>
>/updateUsermess  // 更新个人信息
